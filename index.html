PASATIEMPO
-- En Cascada --

Instrucciones

Identifique la primera palabra y la última de cada bloque con las pistas que se dan.

A continuación, trate de descubrir las palabras intermedias. Para lograrlo, cambie una letra de la primera palabra para obtener la próxima y, después, altere el orden de una o varias letras para encontrar la siguiente.

Siga así, sucesivamente, hasta que logre completar ambos casilleros. Todas las palabras intermedias deben tener significado.

Pistas

  1. Familia en Escocia

  2. Tristeza y dolor por algo

  3. Terminó algo definitivamente

  4. El que torea
