Sobre nosotros

Somos un grupo de estudiantes de Ingeniería Informática que se inicia en el mundo de las Aplicaciones Web tanto Front-end como Back-end y este es nuestro proyecto final de curso realizado sobre Node.js utilizando Bootstrap para realizar las interfaces responsivas.

Se pide desarrollar una aplicación web que permita gestionar y disfrutar de un kiosco virtual en el que los administradores diseñan y venden albumes de kromos para los socios que pueden comprar estos kromos jugando a nuestros propios videojuegos.

sobre onkiskromos

Onkiskromos es una aplicación web Open Source realizada usando la tecnología Node.js en su versión 10.24.0 en el lado del servidor, Bootstrap para el desarrollo de las interfaces y MySQL como gestor de bases de datos para gestionar la persistencia.
